# tezai

Production AI SaaS for thesis and academic document work. Live at **[tezai.com.tr](https://www.tezai.com.tr/)**.

## What it does

Upload a document and get AI-assisted analysis and writing support. Built for real users and paid plans, not a demo.

## Stack

- **Frontend:** Next.js, React, TypeScript
- **AI:** OpenAI, Anthropic Claude, Google Gemini (multi-model)
- **Data:** Supabase, Vercel Postgres
- **Infra:** Upstash (Redis, rate limiting, queues), Vercel
- **Documents:** PDF and Word parsing (pdf-parse, mammoth)
- **Payments:** iyzico

## Status

Live and in production. Source is shared for portfolio reference.
